Documentation

Component Input(s)

Input Type Description
[data] Array Table data array
[settings] ITableSettings Table configuration object
[extendedBody] Angular TemplateRef (ng-template) Provided template can be used as a part of the table all rows.
[className] string The string that is applied as a custom class of the table
[paginationTemplate] Angular TemplateRef (ng-template) It is possible to add your own pagination component

Component Output(s)

Output Type Description
(pageChanged) number Triggers when a page changes
(filter) IFilterConfig Triggers when filtering happens
(sort) ISortConfig Triggers when sorting happens

ITableSettings

Name Type Description
columns IColumns It needs to describe the list of table columns and their behavior
trackBy string It provides a track by property that will be used in *ngFor expression
head ITableHead Using this property it is possible to hide the table head or stick it to the top
pagination IPagination Add and describe pagination for the table
rowClassFunction Function it is used for adding custom classes for the table rows

IColumns

Name Type Description
[name of the property] IColumn The Name can be any valid json property that is used inside of the array of the objects

IColumn

Name Type Description
title string | TComponent Title that will be added as a table cell value (th)
sortable boolean Set possibility to order the table by this column
sortFunction Function If the column has some specific sorting strategy you can specify a custom function
property Function To be able to render a complex object it is possible to provide a custom function that generates it
filter boolean | IFilterSettings Add filter for this column
responsive IResponsiveSetting[] It describes the list of breakpoints and how to transform the table columns under media queries conditions

IComponent

Name Type Description
component Angular Component Any Angular component which is also declared in entryComponent section of its module
componentOnInit Function Within this function it is possible to set all necessary properties and subscribe to the component outputs
componentOnUpdate Function When the value of the table changes, this function will be called in case if you need to manually update the component value or run the detect changes
componentOnDestroy Function It is called when the component calls ngOnDestroy hook. Here you can clean everything related to this component, unsubscribe from the events and etc.

IFilterSettings

Name Type Description
type FilterTypeControl | IComponent Type of the filter. You can set different type of the filters or provide your custom component
list ITitleValue[] The list of possible options, it can be used for FilterTypeControl.select
placeholder string The placeholder for default filter input controls
filterFunction Function The custom filtering function can be used if it is necessary

IResponsiveSetting

Name Type Description
media MediaQueryList Standard matchMedia functionality
column number Number of the column where this column will be moved under provide media query condition
label string | boolean The label that describes this column. If set to true then the title of the column will be used
